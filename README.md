# API-KG

Static API knowledge graph resource built from ToolBench-style JSON catalogs (CIKM resource paper draft).

## Repository contents

| Path | Description |
|------|-------------|
| `generated_stats/category_stats.csv` | Per-category node/edge counts for **50** ToolLLM marketplace partitions (batch build under `*_sample_v4`). |

## Scale (spotlight)

Representative exemplar categories cited in the paper:

| Category | Nodes | Edges |
|----------|------:|------:|
| Jobs | 92 | 227 |
| Data | 7,734 | 76,005 |

Full per-category statistics are in `generated_stats/category_stats.csv`.
